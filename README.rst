############
ucsd-bigdata
############

************
Installation
************

Install via pip:

::

    $ pip install git+https://github.com/mas-dse/UCSD_BigData_Scripts.git

Install from source:

::

    $ git clone https://github.com/mas-dse/UCSD_BigData_Scripts.git
    $ cd UCSD_BigData_Scripts
    $ python setup.py install

*******
Removal
*******

Remove via pip:

::
    $ pip uninstall ucsd-bigdata -y
